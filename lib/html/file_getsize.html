size_t file_getsize

(char* name)

Stand-alone routine to get a file's size.

Documentation

Stand-alone routine to get a file's size. If the file specified exists, its size is returned. Note that a file may be empty, in which case this fails miserably. This is fine for file_new() in Xiqual, however. Reading in an empty file is not necessary when you can just overwrite.
