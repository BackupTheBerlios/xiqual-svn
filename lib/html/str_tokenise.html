strtoken* str_tokenise

(char* s,

Create a tokeniser object from a string.

Documentation

Create a tokeniser object from a string. The supplied string is copied, and this copy is modified with NULL bytes in place of its delimiters. This function does a better job than strtok() and strtok_r(). This function is thread-safe, hopefully.
