char* strsplit

(char* text)

Split a variable-setting string.

Documentation

Split a variable-setting string. Given a string containing two words, separated by eiher whitespace or an equals sign, it returns a pointer to the second word (value), after clearing the spaces between.

The original string is modified so that the pointer given only points to the first word.

Given the string: "var = val" the original pointer will point to "var" and the returned pointer to "val".

All whitespace around the variable and value is stripped.

Parameters:
text - A string
Returns:
Pointer to the second word
