void net_free_server

(SelectServer* server)

Release all resources related to a Xiqual select() server.

Documentation

Release all resources related to a Xiqual select() server. This shuts down the sockets, calling each socket's onclose() hook if available, and cleaning up Win32 sockets if compiled for Win32.
