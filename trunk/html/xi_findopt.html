ulong xi_findopt

(xiopts* opts, char* name)

Find option and return flags.

Documentation

Find option and return flags. Looks for an option in a user-specified struct xiopts. The returned ulong will be set to LOT_FLAG, LOT_KEYWORD, LOT_VALUE or LOT_FLOAT.
Parameters:
- opts User-supplied xiopts.
- name String containing a word or character used as option.
Returns:
ulong containing type of option, NULL if not found.
