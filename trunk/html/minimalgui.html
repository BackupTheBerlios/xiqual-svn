Xiqual 0.9.1

minimalgui

A very minimal example of LUI use. This example shows how to start and end a Xiqual program with LUI gadgets. The user interface runs itself once the gadgets are created, thanks to xi_main(). One button is created with a hook to set an exit flag. Xiqual has no way to know when to quit otherwise.
